# 사업단 경비 처리 업무 매뉴얼

## 경비 처리 전체 프로세스

```
사전 기획 → 내부 품의 → 집행/계약 → 증빙 수집 → 결의서 작성 → 결재/지급 → 정산/보관
```

## 결의서 유형

| 유형 | 용도 | 예시 |
|------|------|------|
| 지출결의서 | 물품 구매, 용역비, 카드대금 등 지출 | 연구재료비, 장비 구매, 컨설팅 |
| 수입결의서 | 연구비 입금, 환불 수입 | 병원대응자금, 카드 환불 |
| 대체결의서 | 계정 간 대체, 인건비 | 인건비, 퇴직충당금, 이자수입 |
